# Ted Card App
Ted Card uygulaması Ted kartlı olan kişilerin yararlana bilecekleri kampanyaları listeler ve kampanya hakkında bilgi verir.

# Uygulamada Kullanılanlar
- Retrofit
- Dagger - Hilt
- Glide
- Kotlin Navigation
- Coroutines
- MVVM
- State Flow
